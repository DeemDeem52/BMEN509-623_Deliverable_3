# BMEN 509/623 — Deliverable 3: Ultrasound Imaging

## 📅 Due Date: March 16, 2026

## Overview

This deliverable covers ultrasound physics and Doppler imaging. You will analyze acoustic wave propagation, explore tissue-ultrasound interactions, process Doppler signals, and identify common ultrasound artifacts in clinical scenarios.

## Getting Started

1. **Read the instructions**: Open `deliverable-03-instructions.pdf` for complete assignment details, learning objectives, and submission requirements.

2. **Complete the starter notebook**: Open `deliverable-03-starter.ipynb` in Jupyter and work through the exercises. The notebook contains scaffolded code cells and questions to guide your analysis.

3. **Explore the data**: The `data/` folder contains ultrasound phantoms, Doppler signals, and tissue property data needed for the exercises.

4. **Submit your work**: Upload your completed notebook to D2L by the due date.
